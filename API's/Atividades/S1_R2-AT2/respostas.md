## Qual a diferença entre biblioteca e framework?

### Biblioteca
* É um conjunto de funções e ferramentas que podem ser chamadas quando necessárias. Com elas, é possível ter o controle do fluxo da aplicação. Exemplo: Math (para cálculos avançados de matemática). É como pegar uma ferramenta epecífica dentro de uma caixa cheia de ferramentas.

### Framework
* É um conjunto de regras e estruturas pré-definidas para facilitar o desenvolvimento de um projeto. O framework dita como a aplicação deve ser estruturada. Exemplo: Express.js (para criar servidores locais). Porém deve se seguir todos os parâmentros desse framework.
